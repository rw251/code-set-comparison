Instructions

This tool allows you to quickly compare two different clinical code sets. Simply paste one code set in the left hand box, and one code set in the right hand box.

The tool assumes that you are pasting tabular data (or tab-separated data), and that either:

  • there are 2 columns: the first is the clinical code, and the second is the description, or
  • there are more than 2 columns but at least one is a clinical code and at least one is a description
